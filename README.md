# 🏥 Hospital Emergency Room Analytics Dashboard

## Project Overview

Developed a **3-page interactive Power BI dashboard** to analyze Emergency Room (ER) operations and patient flow. The dashboard provides insights into patient volume, wait times, satisfaction scores, admission trends, referrals, and demographics, helping stakeholders monitor hospital performance and improve operational efficiency.

## Key Highlights

* Analyzed **9,216 patient records** from **April 2023 to October 2024** to evaluate Emergency Room performance and patient flow.
* Built KPI dashboards to monitor **Total Patients (9,216)**, **Average Wait Time (35.3 Minutes)**, **Patient Satisfaction Score (4.99/5)**, and **Patients Referred (3,816)**.
* Developed interactive visualizations for **admission status, age groups, gender distribution, patient race, department referrals, and day-hour patient trends** using Power BI, DAX, and Power Query.

## Dashboard Pages

### 1. Monthly View

* Monthly performance tracking
* KPI monitoring
* Demographic analysis
* Referral analysis
* Patient flow analysis

### 2. Consolidated View

* Overall hospital performance
* Long-term trend analysis
* Operational insights

### 3. Patient Details

* Detailed patient-level reporting
* Admission and referral information
* Wait time tracking
